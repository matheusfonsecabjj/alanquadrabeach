## 🚀 Como publicar no GitHub Pages

1. Crie um repositório no GitHub (ex: `alan-quadras-beach`)
2. Faça upload de **todos** os arquivos desta pasta
3. Vá em **Settings → Pages**
4. Em *Branch*, selecione `main` e pasta `/ (root)`
5. Clique em **Save**
6. Seu site estará disponível em: `https://seu-usuario.github.io/alan-quadras-beach`
